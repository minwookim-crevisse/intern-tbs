# 사업지원팀 인턴 면접 과제 문제 출제의도 및 해설

## 📋 문제 1: SaaS 서비스 사용료 정산 오류 검수

### 1. 출제의도

이 문제를 통해 평가하고자 하는 핵심 역량은 다음과 같습니다.

1. **정확성(Accuracy)**: 복잡한 정산 데이터에서 오류를 정확하게 식별하고 올바른 금액을 산출하는 능력
2. **논리적 사고력(Logical Thinking)**: 변동이력, 구독대장, 요청서 등 여러 자료 간의 관계를 파악하고 일관성을 검증하는 능력
3. **문서 해독력(Document Comprehension)**: 정산 매뉴얼과 참고자료를 읽고 규칙을 정확히 이해하여 적용하는 능력
4. **일할계산 능력(Proration Calculation)**: 월 중 발생한 변경사항에 대해 일할계산 공식을 적용하는 수리능력
5. **프로세스 개선 마인드(Process Improvement)**: 오류 재발 방지를 위한 검수 방법을 체계적으로 제시하는 능력

---

### 2. 난이도

**난이도: 중상 (中上)**

**이유:**
- **복합성**: 3가지 독립적인 오류 유형(상품 변경, 일할계산, 이월잔액)을 동시에 찾아야 함
- **계산 난이도**: 일할계산이 필요하며, 일단가 산출 및 일수 계산이 포함됨
- **자료의 분산**: 정답을 도출하기 위해 5개 이상의 참고자료를 교차 검증해야 함
- **실무 맥락**: 단순 계산을 넘어 실무적 정산 프로세스 이해가 필요함
- **검수 방법 제시**: 단순히 정답을 찾는 것을 넘어 오류 방지 방법론까지 요구함

**인턴 수준 적합성**: 인턴에게 요구되는 꼼꼼함과 기본 수리능력을 평가하되, 과도하게 복잡한 업무지식을 요구하지 않아 적절함

---

### 4. 모범답안 해설

#### 4.1 D1 펀드레이징 에디션 오류 분석

**문제 상황:**
- 정산서에 FR-0001(33,000원)이 기재되어 있음
- [참고자료 #2-1] 상품주문 요청서에서 FR-0002로 변경 요청 확인

**정답 도출 과정:**
1. [상품주문 요청서]에서 FR-0001 → FR-0002 변경 확인
2. [상품대장]에서 FR-0002의 1개월 단가 확인: 55,000원
3. 구독기간이 1개월이므로 월정액 55,000원 적용

**정답: 55,000원**

#### 4.2 D4 확장사용량 오류 분석

**문제 상황:**
- 정산서에 확장사용량이 33,000원으로 기재됨
- [참고자료 #2-2]에서 3월 14일부터 1개 추가 확인

**정답 도출 과정:**
1. 기존 사용량: 33,000원 (1개 × 월정액)
2. 추가 사용량 (3월 14일~31일, 17일간):
   - 일단가: 1,085원
   - 17일 × 1,085원 = 18,445원
3. 총합: 33,000 + 18,445 = **51,445원**

#### 4.3 L1 메시지 충전금 기초잔액 오류 분석

**문제 상황:**
- 3월 기초가 10,000,000원으로 기재
- 2월 기말은 2,500,000원

**정산 규칙:**
- "전월 기말 = 당월 기초" (이월 원칙)

**정답: 2,500,000원**

## 📊 문제 2: 미디어 사업 사용료 정산서 작성

### 1. 출제의도

1. **데이터 집계 능력(Data Aggregation)**: 여러 광고 채널의 집행비용을 정확하게 합산하는 능력
2. **수수료율 적용(Commission Calculation)**: 채널별로 다른 수수료율을 정확히 적용하여 매출을 산출하는 능력
3. **세무 기초 지식(Tax Knowledge)**: VAT 포함/제외 금액을 구분하고 공급가를 계산하는 능력
4. **자금관리 이해(Fund Management)**: 선수금과 추가입금을 반영하여 세금계산서 발행금액을 산정하는 능력
5. **커뮤니케이션 정보 활용(Communication Skills)**: 담당자와의 소통이력에서 필요한 정보를 추출하는 능력

---

### 2. 난이도

**난이도: 중 (中)**

**이유:**
- 수수료율 계산은 단순하나 5개 채널 처리 필요
- K사, N사의 VAT 포함 금액 환산 과정 추가
- 선수금 개념과 세금계산서 발행 로직 이해 필요

---

### 3. 평가 요소별 배점 기준 (총 100점)

| 평가 항목 | 정답 | 배점 |
|----------|------|------|
| 광고소재 합계 | 5,390,000원 | 15점 |
| G사 매출 | 5,582,793원 | 10점 |
| M사 매출 | 19,583,643원 | 10점 |
| K사 매출 (VAT 환산) | 4,980,344원 | 10점 |
| X사 매출 | 3,463,270원 | 10점 |
| N사 매출 (VAT 환산) | 1,597,090원 | 10점 |
| Total | 40,597,140원 | 10점 |
| Total VAT | 44,656,854원 | 10점 |
| 세금계산서 발행금액 | 26,000,000원 | 15점 |

---

### 4. 모범답안 해설

#### 4.1 광고소재 매출 집계

**출처**: [참고자료 #3] 소통이력 (이력 3)

```
광고소재 A: 1,890,000원
광고소재 B: 3,500,000원
───────────────────────
합계: 5,390,000원 (VAT 제외)
```

#### 4.2 광고집행 매출 산출

**수수료율 정보** (참고자료 #5):
| 채널 | 수수료율 | VAT 포함 여부 |
|------|---------|--------------|
| G사 | 15% | N (제외) |
| M사 | 15% | N (제외) |
| K사 | 10% | Y (포함) |
| X사 | 10% | N (제외) |
| N사 | 10% | Y (포함) |

**계산 예시 (K사, N사):**
```
K사: 집행비용(VAT포함) ÷ 1.1 = 공급가
     공급가 × 수수료율 = 수수료
     공급가 + 수수료 = 매출
```

#### 4.3 세금계산서 발행금액

**선수금 계산:**
- 2월 이후 잔액: 8,656,853원 (이력 1)
- 3월 추가 입금: 10,000,000원 (이력 2)
- 합계: 18,656,853원

**발행금액:**
```
Total VAT - 선수금 = 세금계산서 발행금액
44,656,854 - 18,656,853 = 26,000,001원
≈ 26,000,000원 (천원 단위 반올림)
```
